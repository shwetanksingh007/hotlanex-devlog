# 🎮 Hotlanex

## 🚀 Overview
Hotlanex is a high-speed racing game where the player escapes from AI enemies while collecting resources like fuel, coins, and flags. The game focuses on intelligent enemy behavior and dynamic gameplay.

---

## 🎯 Features
- 🧠 AI enemies with improved pathfinding (A*)
- ⛽ Fuel-based mechanics
- 🚗 Car selection system
- 🪙 Collectibles (coins, flags, powerups)
- 🔥 Real-time gameplay decisions
